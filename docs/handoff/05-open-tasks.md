# 05 残タスク（次にやること）

## Phase 5-3: テナント管理UI（最小スコープ推奨）
「作成/編集」まで一気にやると重いので、まずは *運用価値が高い最小* から。

### 推奨スコープ（MVP+）
1) **現在のテナント表示（ナビ）**
2) **テナント切り替えUI**
   - ユーザーが所属するテナント一覧（ドロップダウン）
   - 選択すると `/t/{tenant}/dashboard` などへ遷移
3) アクセス制御
   - 所属していないtenantへ遷移しようとしたら404

### オプション（後回しでOK）
- テナント作成/編集（管理者のみ）
- テナント招待/所属管理（ユーザー管理と絡むため要件固定が必要）
